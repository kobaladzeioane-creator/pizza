# pizza-and-salat
# 🍕 Pizza & Salad Project 🥗

ეს არის მარტივი და ეფექტური აპლიკაცია, რომელიც განკუთვნილია პიცებისა და სალათების მენიუს სამართავად. პროექტი კითხულობს და ამუშავებს მონაცემებს ადგილობრივი ბაზიდან (data.json) და გამოაქვს ინტერფეისში.

---

## 🛠️ ტექნოლოგიები (Tech Stack)
* *JavaScript (ES6+)* - ლოგიკისა და მონაცემების მანიპულაციისთვის.
* *Node.js* - პროექტის გარემოსა და პაკეტების მენეჯმენტისთვის.
* *JSON* - პროდუქტების მონაცემების შესანახად.

---

## 📂 ფაილების სტრუქტურა
* script.js — მთავარი სკრიპტი, სადაც იწერება პიცებისა და სალათების წამოღების, გაფილტვრისა და დამუშავების ლოგიკა.
* data.json — მონაცემთა ბაზა, სადაც ინახება ინფორმაცია პროდუქტებზე (სახელი, ფასი, ინგრედიენტები, კატეგორია).
* package.json — პროექტის კონფიგურაცია და დამოკიდებულებები (Dependencies).
